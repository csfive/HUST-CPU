## 5 输入编码器

:::tip
根据 Excel 表得出下列布尔表达式，在电路分析中填入表达式，自动生成电路。

```
Out1 = In5+In3&~In4&~In5+In1&~In2&~In3&~In4&~In5
Out2 = In3&~In4&~In5+In2&~In3&~In4&~In5
Out3 = In5+In4&~In5
```

:::

## 2 路选择器（1 位）

:::tip
1 位 2 路选择器的布尔表达式可以写成如下形式，根据表达式连接电路。

```
Out = X0 ~Sel + X1 Sel
```
:::

![2路选择器](/1/2路选择器1位.png)

## 2 路选择器（16 位）

:::tip
16 个 1 位通过分线器并发成 1 个 16 位的 2 路选择器。

自动测试显示 PASS 表示测试通过，电路正确。
:::
